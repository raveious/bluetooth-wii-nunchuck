# PCB Design

## Design Requirements

- Has to fit in the case of the original, Nintendo Wii Nunchuck.
- Battery powered.
- Rechargable.
- Bluetooth with a pretty decent range.
- Power switch.

Extras:
- Act as a HID with 5 axes.
- USB reprogrammable.
- USB debug.
- USB Charging.
- Status LEDs.
