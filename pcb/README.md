# PCB Design

## Design Requirements

- [x] Has to fit in the case of the original, Nintendo Wii Nunchuck.
- [x] Battery powered.
- [x] Rechargable.
- [x] Bluetooth with a pretty decent range.
- [x] Power switch.

Extras:
- [ ] Act as a HID with 5 axes.
- [x] USB reprogrammable.
- [x] USB Charging.
- [x] Status LEDs.
- [ ] Report battery status over Bluetooth.
 - Use MAX17043
